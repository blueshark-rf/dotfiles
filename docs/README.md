# Client Build and Dotfiles SOP Collection

This repository contains standard operating procedures for client build configuration, automation, and maintenance.

| SOP | Title | Description |
|-----|--------|-------------|
| SOP01 | Homebrew Installation and PATH Standardisation | Establishes consistent tooling and PATH ordering. |
| SOP02 | Git and Delta Configuration | Configures Git with delta for coloured diffs. |
| SOP03 | SSH, GitHub Authentication and Chezmoi Remote Setup | Handles SSH key and remote GitHub auth. |
| SOP04 | Python and Pipx Standardisation | Unifies Python interpreter and pipx environment. |
| SOP05 | Dotfiles Audit and Reporting | Automates inspection of all dotfiles. |
| SOP06 | Chromaterm Configuration and Colour Management | Configures terminal colour themes and filters. |
| SOP07 | Client Build Index and Maintenance | Master index for client setup and maintenance. |
| SOP08 | Client Rebuild from Dotfiles Repository | Automates complete workstation rebuild via chezmoi. |
